Usage

Paste the seed data into the text area and submit it.
Choose appropriate options for each field.
The name field specifies the name of the JSON file.
Name of field specified between square brackets will be replaced by the value of the field.

Eg:- If name is given as
RoleEntity%3a%3a[roleEntityId]
the [roleEntityId] will be replaced by the value of role entity id.

Submit it to download the JSON files.
