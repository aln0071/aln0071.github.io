Google Meet Manager

This chrome extension automatically disconnects Google Meet if number of attendees go below the limit set by user. The user must install the Google Chrome Extension and set the limit after starting the meet. I have added the Precision Mode to the new Version 1.1 which can distinguish between duplicate users in the meet to get the actual number of users.

Teams Call Manager

This chrome extension automatically disconnects Microsoft Teams call if number of attendees go below the limit set by user. The user should start the call from Teams Web Application, and then open the extension to set a limit.

Header Changer

This is a privacy enhancement tool made available for both Chrome and Firefox browsers. It allows the user to edit the title displayed on the current tab. It is useful when you switch tabs, and you don't want others to know what is on the other tab by reading the tab title.

Precision Mode

The extension automatically switches to the Precision Mode when the user keeps the 'Meeting details' tab open. By default, the number of attendees reported by the Google Meet Web Application can contain duplicates. These duplicates occure when someone starts sharing their screen or when a person joins the meet from multiple devices or tabs. With the new update, our Chrome Extension can determine the actual number of attendees who are there in the meet without considering the duplicates. Please note that the Precision Mode is active only when you keep the 'Meeting details' tab open. To open the 'Meeting details' tab, please follow these steps:
Option 1: On the top right corner of your screen, you can see a toolbar as shown in the below image. Please click on any of the encircled icon in that toolbar to open the meeting details tab.
Option 2: In the extension's popup, click on the 'Meeting details' button, which will open the meeting details tab for you.
